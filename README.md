# Hyper-Local Civic Transparency Engine

## Problem
Citizens often see development projects but lack clarity on:
- Budget usage
- Completion status
- Civic impact

## Solution
A geo-fencing based system that delivers transparent, location-aware
notifications explaining public development work.

## Tech Stack
- Node.js + Express
- PostgreSQL
- Prisma ORM
- Geo-fencing (Haversine formula)

## Core Features
- Project geo-fencing
- Location validation
- Context-aware notifications
- Engagement analytics
- Privacy-first design (no tracking)

## Backend Architecture
- Modular DDD structure
- REST APIs
- Stateless services
